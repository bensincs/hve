# Developer Tools

A collection of AI-powered tools and frameworks to enhance your development workflow.

## GitHub Copilot

AI-powered code completion and chat assistant integrated into VS Code. Provides inline suggestions, natural language chat, and autonomous coding capabilities.

[Learn more →](./copilot.md)

## GitHub MCP Server

Connect AI tools directly to GitHub's platform with the Model Context Protocol server. Enables AI agents to read repositories, manage issues and PRs, and automate workflows.

[Learn more →](./github-mcp-server.md)

## Spec Kit

Toolkit for spec-driven development that flips traditional software development by making specifications executable and directly generating working implementations.

[Learn more →](./speckit.md)
